✍️ Deskripsi Singkat

Saya adalah Gregorius Craig Christian Ishak, siswa kelas 11.S7 di Kolese Kanisius Jakarta. Di luar kelas, saya dikenal sebagai sosok yang aktif secara fisik. Bagi saya, olahraga bukan sekadar hobi, melainkan bagian dari diri saya. Lari, sepak bola, dan berbagai aktivitas fisik lainnya melatih kedisiplinan dan semangat kompetitif yang juga saya bawa ke dalam belajar. Dalam mata pelajaran TIK semester ini, saya belajar bahwa membuat website tidak jauh berbeda dari olahraga, butuh latihan konsisten dan kemauan untuk terus berkembang. Saya mempelajari HTML sebagai fondasi, CSS sebagai pelengkap, dan JavaScript sebagai alat untuk membuat sesuatu menjadi interaktif.

Melalui pelajaran TIK, saya ingin memahami bagaimana teknologi bekerja, dari website yang saya kunjungi setiap hari hingga aplikasi yang saya gunakan untuk berolahraga.
